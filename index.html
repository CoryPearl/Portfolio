Multiplayer Minecraft:
In my class we had a open ended assignment, I was thinking of posible games I could make and 2D minecraft poped into my head. So I started making a web based 2D Minecraft when I realized I had never made a 3D game before. I thought this would be a great challenge to imporove my javascript skills. So using three.js (a 3D library made by MIT) I made a randomly generated world with sand, tress, dirt and rocks where you could just break blocks and place blocks. How ever we still had quite some time left for the project. So I took it a step further and made it multiplayer using node.js. Using sockets I manged to make a full fledged multiplayer game with a chat, player modles that point in right directions, player names and online player display. Some smaller things I incorperated were music with a mute and unmute button, fps display, flying mode and a coordinate system.
