CodeSync (current project):
This year I'm taking IB computer science. The way it works is you do something called an 'IA' which is a year long project. So I decided to try to create my own version of Replit that allows you to code online with a team in java, python and html/css/js. So far I've created sign in, account creation, log out and now I'm moving onto creating collaborative spaces. I'm also using a better storage systems than when I made ByteNet (scroll down to see), I do not have access to a free SQL server so when I made ByteNet I stored everything in separate text files but this time I'm using JSON to store and access it easier and cleaner.

Multiplayer Minecraft:
In my class we had an open -ended assignment. I was thinking of possible games I could make and 2D minecraft popped into my head. So I started making a web based 2D Minecraft when I realized I had never made a 3D game before. I thought this would be a great challenge to improve my javascript skills. So using three.js (a 3D library made by MIT) I made a randomly generated world with sand, trees, dirt and rocks where you could just break blocks and place blocks. However we still had quite some time left for the project. So I took it a step further and made it multiplayer using node.js. Using sockets I managed to make a full fledged multiplayer game with a chat, player models that point in right directions, player names and online player display. Some smaller things I incorporated were music with a mute and unmute button, fps display, flying mode and a coordinate system.

ByteNet:
ByteNet is my biggest project yet. It is a social media website based around computer science. It has basic functionality of social media, you can friend people and message them, post text, images and code snippets, read the news, and create an account, sign in and change all your info including profile picture and bio. This project involved a lot of things I had never done before and a lot of problem solving. This is also my first full stack web app I've made. In doing this I also learned how to setup a ubuntu web server using DartNode, setup a domain name and get a HTTPS certificate. I also learned how to encrypt sensitive information such as passwords.
